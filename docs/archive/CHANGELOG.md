## Older Updates

The full changelog contains many more entries dating back to 2020. Due to the length, only recent entries are shown here.

### 2025 Updates Summary
- Multiple muapDotNET and MDSound version updates
- PartCounter improvements for various drivers (mucomDotNET, PMD)
- IDE enhancements and bug fixes
- Core functionality improvements for OPN series chips

### 2024 Updates Summary
- .NET 8 migration for x64 versions
- ZGM and XGM2 format improvements
- POKEY chip support enhancements
- SCCI and SCCI2 support
- Various core chip emulation improvements

### 2023 Updates Summary
- YM2609 chip support
- ZGM format support for various chips (NES, VRC6, DMG, etc.)
- MIDI keyboard support
- MDSound library updates

### 2022-2020 Updates Summary
- Initial support for many sound chips (AY8910, OPN, OPM, OPL, etc.)
- IDE feature additions (PartCounter, Folder Tree, etc.)
- Export functionality (WAV, MP3, VGM, etc.)
- Various bug fixes and performance improvements
