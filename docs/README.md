## Quick Start

1. **Install**: Extract bin.zip and run mml2vgmIDE.exe
2. **Create**: File > New to create a new .gwi MML file
3. **Play**: Press F5 to compile and play
4. **Learn**: Check the MML Commands reference for syntax

## Supported Sound Chips

mml2vgm supports an extensive list of sound chips including:
- **Sega Mega Drive**: YM2612, SN76489
- **PC Engine/TurboGrafx-16**: YM2608, HuC6280
- **Arcade**: YM2151, YM2610B, YM3526, Y8950, C140, C352
- **MSX**: YM2413, YM2203
- **Nintendo**: NES APU, FDS, DMG (Game Boy)
- **Other**: AY8910, OPLL, OPL2, OPL3, QSound, SCCI, K053260, and many more

See [MML_Commands.md](MML_Commands.md) for the complete list.
